## 🚀 Features

* **Real-time Tracking**: Intuitive timer with the signature "Wave" 🌊 interface.
* **Weekly Insights**: Automatic aggregation of tasks by project and day.
* **Dual Mode UI**: Seamless switching between Light and Dark "Fintech" themes.
* **Detailed Analytics**: Grouped historical data with manual entry correction.
